# Sistema de Loja com Django

Desenvolvendo um sistema web de loja virtual desenvolvido com Django. Para aprimorar as habilidades com python e django. O projeto possui autenticação de usuários, painel administrativo, gerenciamento de produtos e integração com API externa para exibição de produtos.

---

## Funcionalidades

### Usuários
- Cadastro de usuários
- Login e logout
- Edição de perfil
- Visualização de compras realizadas
- Avaliação de produtos comprados

### Produtos
- Cadastro de produtos manualmente
- Edição de produtos
- Exclusão de produtos
- Upload de imagens

### Loja
- Listagem de produtos cadastrados no sistema
- Integração com a Fake Store API
- Compra de produtos
- Exibição de preços e disponibilidade

### Dashboard Administrativo
- Gerenciamento de usuários
- Gerenciamento de produtos
- Gerenciamento de categorias
- Gerenciamento de compras
- Gerenciamento de contatos
